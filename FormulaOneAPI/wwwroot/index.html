Hvordan bruke Web API

Introduksjon

Med denne siden viser vi hvordan du bruker Formel 1 API, slik at du får tak i det du trenger!

Vi har to APIer:
  • Drivers brukes for å få tak i alle drivers
  • Teams brukes for å få tak i alle teams

Drivers

Det er totalt 16 drivers i listen, som er sortert med id.

Hvert driver-objekt inneholder denne informasjonen:

Nøkkel Type Beskrivelse
id int ID til fører
name string Navnet til fører
age int Alderen til fører
nationality string Nasjonaliteten til fører
image string (url) Bilde av fører

Det er flere ting man kan gjøre med en driver:

  • POST brukes for å legge til en ny driver i databasen, malen er /api/Drivers
  • PUT brukes for å endre et eksisterende team i databasen, malen er /api/Drivers
  • DELETE brukes for å slette et team fra databasen, malen er /api/Drivers/{id}

  • POST brukes for å legge til bilde av driver i databasen, malen /api/ImageUpload
En driver kan se slik ut:
Hente alle drivers

Malen for å hente alle drivers er api/Drivers.

GET https://localhost:5100/api/Drivers

[
{"id":1,"name":"Max Verstappen","age":26,"nationality":"Netherlands","image":"Verstappen.png"},
{"id":2,"name":"Sergio Perez","age":33,"nationality":"Mexico","image":"Perez.png"},
// ...
]

Hente en driver med id

Malen for å hente en driver med id er api/Drivers/{id}.

GET https://localhost:5100/api/Drivers/1

[
{"id":1,"name":"Max Verstappen","age":26,"nationality":"Netherlands","image":"Verstappen.png"}
]

Hente en driver med navn

Malen for å hente en driver med navn er api/Drivers/GetDriverByName/{name}.

GET http://localhost:5100/api/Drivers/GetDriverByName/Max%20Verstappen

[
{"id":1,"name":"Max Verstappen","age":26,"nationality":"Netherlands","image":"Verstappen.png"}
]

Teams

Det er totalt 6 teams i listen, som er sortert med id.

Hvert team-objekt inneholder denne informasjonen:

Nøkkel Type Beskrivelse
id int ID til teamet
manufacturer string Navnet til teamet
image string (url) Bilde av bilen til teamet
driverone string Fører 1 i teamet
drivertwo string Fører 2 i teamet

Det er flere ting man kan gjøre med et team:

  • POST brukes for å legge til et nytt team i databasen, malen er /api/Teams
  • PUT brukes for å endre et eksisterende team i databasen, malen er /api/Teams/PutUpdateTeam/{manufacturer}
  • DELETE brukes for å slette et team fra databasen, malen er /api/Teams/DeleteTeamByManufacturer/{manufacturer}

  • POST brukes for å legge til bilde av bilen til teamet i databasen, malen /api/ImageUpload/PostUploadCarImage
Hente alle team

Malen for å hente alle team er api/Teams.

GET https://localhost:5100/api/Teams

[
{"id":1,"manufacturer":"Redbull","image":"Redbull.png","driverOne":"Max Verstappen","driverTwo":"Sergio Perez"},
{"id":2,"manufacturer":"Mercedes","image":"Mercedes.png","driverOne":"Lewis Hamilton","driverTwo":"George Russell"},
// ...
]

Hente et team med id

Malen for å hente et team med id er api/Teams/{id}.

GET https://localhost:5100/api/Teams/1

[
{"id":1,"manufacturer":"Redbull","image":"Redbull.png","driverOne":"Max Verstappen","driverTwo":"Sergio Perez"},
]

Hente et team med navn

Malen for å hente et team med navn er api/Teams/GetTeamByManufacturer/{manufacturer}.

GET http://localhost:5100/api/Teams/GetTeamByManufacturer/Redbull

[
{"id":1,"manufacturer":"Redbull","image":"Redbull.png","driverOne":"Max Verstappen","driverTwo":"Sergio Perez"},
]
